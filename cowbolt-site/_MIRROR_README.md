Static mirror of the live cowbolt.com marketing site, captured 2026-05-30.
All pages (home, about, join, terms-and-conditions) + brand assets, icons, CSS, JS, favicons.

Source note: GoDaddy holds cowbolt.com's domain/DNS only (Domains API, no hosting/file access),
so this was captured by mirroring the live deployed static site rather than pulled from a host panel.
This doubles as a backup while cowbolt.com's TLS certificate is expired.
